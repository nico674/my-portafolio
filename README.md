## 📋 Descripción

Landing page de portafolio profesional que muestra mi perfil como estudiante de Ingeniería Electrónica apasionado por el desarrollo frontend. El sitio presenta:

- Hero section con presentación personal
- Áreas de expertise (Frontend, IoT, Prototipado)
- Proyectos destacados (incluyendo colaboración en libro de amplificadores operacionales)
- Sección "Sobre mí" con estadísticas y habilidades técnicas
- Formulario de contacto funcional (simulación)
- Diseño completamente responsivo
